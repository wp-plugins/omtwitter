Plugin for simple periodic backup of Tweets.

== Description ==

Plugin for simple periodic backup of Tweets. Can store your Tweet to separate category or can mix them with your blog
posts. Compare https://twitter.com/OzzyCzech and on my blog: http://www.nabito.net/tweets

== Installation ==

1. Upload the files to the /wp-content/plugins/omTwitter/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to Options -> omTwitter and setup your Twitter account.
